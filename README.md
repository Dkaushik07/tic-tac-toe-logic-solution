# Simple Tic-Tac-Toe: A Mental Logic Tool

# Problem Statement

People require breaks from work activities which become necessary after they experience high stress levels in their office or classroom environments. I created this version of Tic-Tac-Toe to serve that purpose. The Tic-Tac-Toe game provides a brain workout which enables players to think their best. The game of Tic-Tac-Toe helps players develop their strategic thinking skills. Players use Tic-Tac-Toe to achieve mental relief during times of intense stress.

# How it Works

1. The Tic-Tac-Toe game is played on a grid with 3 rows and 3 columns.

2. Players take turns putting their mark either an X or an O on the Tic-Tac-Toe grid by entering a number from 1 to 9.

3. The first player to get 3 of their marks in a row either across, up and down or diagonally wins the Tic-Tac-Toe game.

4. If all of the squares are filled and no one has won the Tic-Tac-Tie game then the game ends in a tie.
