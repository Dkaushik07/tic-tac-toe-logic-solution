# Simple Tic-Tac-Toe: Mental Logic Tool

# Problem Statement
In high-stress environments like classrooms or offices, people often need "micro-breaks" to refresh their minds. This project provides a simple, distraction-free digital version of Tic-Tac-Toe that serves as a quick mental exercise to improve logical thinking and strategy.

# How it Works
1. The game is played on a 3x3 grid.
2. Players take turns entering a number (1 through 9) to place their mark (X or O).
3. The first player to get 3 in a row (horizontally, vertically, or diagonally) wins.
4. If all 9 squares are full and no one has won, the game ends in a tie.

# Tools Used
- Language: Python 3
- Concepts: Functions, Loops, Conditional Logic, and Input Handling.

# Setup Instructions
1. Download the `main.py` file.
2. Open your terminal or command prompt.
3. Run the command: `python main.py`
